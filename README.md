# legalrag3

Minimal workspace for experimenting with the MVP solo agentic workflow.

## Hello World

Open `index.html` in a browser to view the first static page.

## Linear API Helper

Routine Linear updates should use `scripts/linear_cli.py`. See
`docs/linear-api-usage.md` for setup and examples.

## Legal RAG System Description

The technical source document for the planned Legal RAG Assistant is available in
`docs/legal-rag-system-description.md`. It describes the product scope, system
architecture, core workflows, data model, quality metrics, and implementation
phases for future agent-driven development.
